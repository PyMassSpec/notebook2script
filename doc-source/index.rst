Installation
--------------

``pkgname`` can be installed from PyPI or Anaconda.

To install with ``pip``:

.. code-block:: bash

	$ python -m pip install pkgname

To install with ``conda``:

.. code-block:: bash

	$ conda config --add channels http://conda.anaconda.org/domdfcoding
	$ conda install pkgname
